Sauvegarde & Restauration

Cet onglet vous permet de sauvegarder toutes les données de l'application (audits, NC, actions, magasins, utilisateurs…) dans un fichier JSON, et de les restaurer en cas de problème.

Comment faire une sauvegarde ?
Cliquez sur Exporter les données régulièrement (hebdomadaire recommandé). Conservez le fichier JSON dans un endroit sûr (cloud, disque dur…).

Comment restaurer ?
En cas de perte de données, cliquez sur Importer une sauvegarde, sélectionnez votre fichier JSON. Attention : l'import écrase toutes les données actuelles.
